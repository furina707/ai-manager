# AI 管理工具 (AI Manager)

一个用于管理多个 AI CLI 工具的图形化管理器。

## 功能特性

- 🤖 **支持多种 AI 工具**
  - Qwen Code (通义千问代码助手)
  - Gemini CLI (Google Gemini 代码助手)
  - OpenAI Codex (OpenAI 代码工具)

- ⚡ **便捷管理**
  - 一键安装/更新/卸载
  - 快速启动 AI 工具
  - 自动检查版本更新

- 🎨 **现代化界面**
  - 深色模式
  - 简洁直观的操作界面
  - 实时状态显示

## 使用方法

1. 双击运行 `AI 管理工具.pyw`
2. 选择需要管理的 AI 工具
3. 点击对应功能按钮进行操作

## 系统要求

- Windows 系统
- Python 3.8+
- Node.js (用于 npm 包管理)
- 依赖库：`customtkinter`

## 安装依赖

```bash
pip install customtkinter
```
